# CBandit

Common Bandit. Utility functions.

## Installation

```bash
composer require nmg/cbanditphp
```

## Using GlobalClass method

```php
$g = new nmg\CBandit\GlobalClass();
$g->nfo();
```

## Using a simple function from src/CBandit/functions_helper.php

```php
nfo();
```
